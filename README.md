## Install

* Go to `cd xcode-file-template-generator`
* Run `chmod +x run.sh`
* Change `template-text.txt`
> Example: 
> ```swift
>// ==================
>//  Design by ubahwin
>// ==================
>
>import Foundation
>
>```

* Run `./run.sh <template name>`
* Done!
